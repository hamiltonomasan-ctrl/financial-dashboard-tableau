# Financial Dashboard (Tableau)

## Overview
This project is an interactive Tableau dashboard designed to analyse financial data and identify spending trends and patterns.

## Features
- Visualises income, expenses, and savings trends  
- Includes line, bar, and pie charts  
- Highlights category-level spending behaviour  
- Provides insights through interactive filtering  

## Tools Used
- Tableau
